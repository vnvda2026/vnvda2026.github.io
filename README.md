# Conference Talks Mini App

A lightweight React + TypeScript app for browsing conference reports from a TSV feed.

## Features

- TSV cached locally after the first load
- Reload button to force a fresh TSV fetch and overwrite cache
- Sidebar search with comma-separated keywords
- YouTube embed with local playback progress resume
- Local status tags: will watch, watching, completed
- Slide link for each report

## Run

```bash
npm install
npm run dev
```
